Interactive Plot Website

Summary

This Interactive plot website shows how to create a webpage with interactive plots. The approach is to use GitHub Pages, a free website hosting platform integrated with GitHub. Here are some reasons GitHub Pages is good:

  • It is free!
  • The code implementing the webpage is in a GitHub repository, giving you all of GitHub's nice version control features.
  • You can use the same GitHub repo for code, data, and webpage. This makes it a one-stop shop for a research project: When you open-source the code for your paper, you can have a webpage with interactive plots in the same repo and GitHub will host it.
My first experience with GitHub Pages was making a website for a game engine I open-sourced a couple years ago. I liked it a lot and am glad to use it again!

To make the interactive plot below, I used javascript. Specifically, I used Plotly.js, which has a bunch of open-sourced tools for embedding plots in html webpages. For the data, I pushed a dataset (from a research project) to this GitHub repo, which could then be loaded by the javascript code to make the plot.

Making this website was not too difficult. I'm not very good at html or 2 javascript so I had to spend some time debugging, but with a little practice this would probably become very easy, like Latex or Markdown. Overall it was a lot less painful than my previous attempt at making an interactive website (using distill.pub's framework). I'm now seriously considering making an interactive website for the next paper I write.
